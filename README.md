# office financial management system

A desktop application for managing office income and expenses using Python.

# Features
- ➕ Add Income
- ➕ Add Expense
- ✏️ Edit Data
- 🗑️ Delete Data
- 📊 Calculate Total Income and Expenses
- 💾 Save Data to Excel (.xlsx)

# Technologies
- Python
- Tkinter
- OpenPyXL
